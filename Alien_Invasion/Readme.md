# Alien Invasion (外星人入侵)

本项目基于经典教材 **《Python 编程：从入门到实践 (第 2 版)》** 中的项目案例进行开发。在完整实现书本核心功能的基础上，我根据实际需求进行了功能增强与代码重构。

## 📖 项目简介

《外星人入侵》是一款 2D 射击类游戏。玩家控制一艘飞船，左右移动并射击不断降落的外星人群。随着外星人被消灭，游戏等级会提升，难度（速度与得分倍率）也会随之增加。

## ✨ 核心功能

- **飞船射击系统**：支持键盘左右移动控制及空格键射击。
- **动态难度系统**：游戏节奏随关卡提升（Level Up）自动加快。
- **记分系统**：实时显示当前得分、最高分、等级以及剩余飞船数量。
- **持久化排行榜 (新)**：突破了书本原始案例，实现了将历史战绩保存至本地文件 `high_score.txt`。

## 🛠️ 模块化结构说明

本项目严格遵循教材推荐的模块化开发模式，并进行了以下组织：

- `alien_invasion.py`: **主程序入口**。负责初始化游戏资源、控制主循环及调用各模块函数。
- `settings.py`: **配置模块**。集中管理屏幕尺寸、速度参数、颜色及动态缩放系数。
- `game_stats.py`: **统计模块**。负责内存数据的管理以及排行榜的 **读取与保存** 逻辑。
- `game_function.py`: **逻辑模块**。处理所有事件响应、子弹/外星人更新及碰撞检测逻辑。
- `ship.py`, `alien.py`, `bullet.py`: **精灵模块**。定义了游戏中各实体的属性与行为。
- `score_board.py` & `button.py`: **UI 模块**。负责将文本和按钮信息渲染为屏幕图像。

## 🚀 个人特色功能 (Custom Enhancements)

相比于《Python 编程：从入门到实践》的原始版本，本项目独立开发并集成了以下进阶功能：

### 玩家身份与动态交互

- **需求实现**：在游戏初始化阶段，通过控制台实时获取玩家姓名。
- **技术点**：将外部输入流（`input`）与 `GameStats` 类的初始化逻辑相结合。

### 多用户排行榜系统

- **数据持久化**：打破了原书“最高分随程序关闭而重置”的局限。
- **智能算法**：
  - **读取**：启动时自动解析 `high_score.txt`，支持异常数据清洗（如处理空行或格式错误）。
  - **排序**：利用 `Lambda` 表达式对所有历史成绩进行降序排列。
  - **更新**：游戏结束时自动比对新分数，仅保留前 10 名，确保数据的精简与高效。

### 安全退出与自动存档

- **需求实现**：优化了 `Q` 键及窗口关闭逻辑。
- **技术点**：在 `sys.exit()` 执行前强制触发 `save_scores()` 钩子函数，确保玩家的每一分心血都能被正确记录。

### 代码健壮性优化

- **异常防御**：对文件读取过程增加了防错处理，确保即使 `high_score.txt` 被意外修改，游戏依然能以默认配置正常启动。
